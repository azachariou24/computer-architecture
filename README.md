## Overview

This repository contains educational implementations exploring fundamental topics in **Computer Architecture** through carefully designed programming projects.

Each project focuses on a specific architectural concept while emphasizing low-level programming, mathematical computation, and the relationship between hardware architecture and software execution.

The repository is intended as a collection of well-documented educational projects that demonstrate both practical implementation techniques and the theoretical principles underlying modern computer architectures.

## Projects

| Project | Description |
|----------|-------------|
| [**Taylor Series in MIPS Assembly**](src/taylor-series-mips/) | Recursive and iterative approximation of **ln(1+x)** using the MIPS Floating-Point Coprocessor. |

## Educational Objectives

The primary objective of this repository is to develop a deeper understanding of computer architecture through practical implementations.

Each project aims to connect theoretical concepts with low-level programming techniques, providing educational examples that emphasize correctness, readability, and architectural understanding.

The repository is intended to serve as a long-term educational collection that will continue to grow as new computer architecture topics are explored.
